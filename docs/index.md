## Working areas

- `inbox/` is for unclassified and developing notes.
- `guides/` is for stable instructions about using or extending MathLab.
- `assets/` contains diagrams, HTML, and other supporting files.
- `archive/` contains superseded drafts and historical material.

When a note changes categories, move it rather than maintaining copies in two
places.
