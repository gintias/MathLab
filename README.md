# MathLab

MathLab is a personal Python project for exploring and implementing mathematical
objects, logic, and supporting workspace machinery.

## Mathematical reference

The mathematical ground source of truth for the project is
[`THEORY_ATLAS.md`](docs/concepts/mathematics/THEORY_ATLAS.md). Use the Atlas as
the authoritative reference for mathematical claims, distinctions, and validity.
It is not an implementation architecture specification; package structure, APIs,
and implementation order remain project-design decisions unless explicitly
settled elsewhere.

## Project layout

- `logic/` — logic and recursion-related code
- `objects/` — mathematical object implementations
- `workspace/` — project workspace and registry machinery
- `docs/` — project notes and documentation; this directory is also an
  Obsidian vault

Start with the [documentation index](docs/index.md).
